# WarmAPI Documentation
API that communicates with the Greenhouse Harvest API.
Used in the OLX Challenger to get candidates information.

## Endpoints

|Type |  Endpoint | Description  |
|---|---|---|
|**GET** |/candidate    | Gets all Candidates|
|**GET** |/candidate/:id| Gets a candidate via id|
|**POST**|/candidate    | Posts an email via a requests body to get a candidate back via the Greenhouse Harvest API.|
